# Xin chào 👋 Tôi là TinDev

Tôi là một developer đam mê **lập trình, tự động hóa, API và phát triển web**.

🚀 Tôi thích xây dựng những công cụ hữu ích, giúp công việc nhanh hơn, thông minh hơn và tiện lợi hơn.

🛠️ Hiện tại tôi tập trung vào:
`Bots` • `APIs` • `Automation` • `Web` • `Tools`

Tôi luôn học hỏi, luôn code và luôn cải thiện mỗi ngày.

> Ý tưởng chỉ là bắt đầu — code biến nó thành sản phẩm thật.
